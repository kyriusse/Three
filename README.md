# Three

**Three** est un module de simulation généraliste pour modéliser les interactions entre économie, écosystèmes et événements.

Ce dépôt contient une version structurée de la spécification fonctionnelle, destinée à servir de base de développement pour:
- des simulations personnalisées,
- des outils spécialisés (Inflation, Évolution, Écosystème),
- un futur site web NSI.

## Contenu

- [`docs/THREE_SIMULATION_SPEC.md`](docs/THREE_SIMULATION_SPEC.md): spécification fonctionnelle consolidée (version exploitable pour implémentation).

## Vision

L’utilisateur peut:
- créer jusqu’à 3 univers de simulation,
- définir des objets, des liaisons, des événements et des patterns,
- lancer des projections temporelles,
- observer des impacts économiques et écologiques propagés dans un réseau.
